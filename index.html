Ementa & Competências

Conforme o Projeto Pedagógico do Curso — FATEC Ourinhos / Centro Paula Souza.

📋 Conteúdos da Ementa

  • Software multiplataforma: Mobile, Web e Desktop
  • Sistemas Distribuídos — concorrência, openness, escalabilidade
  • Protocolos de Mensageria — intercomunicação de sistemas
  • Brokers de recepção de dados (MQTT, RabbitMQ, Kafka)
  • Armazenamento de dados em grande escala
  • Data Streaming — mineração de dados em tempo real
  • Utilização de APIs (REST, GraphQL, WebSocket)
  • Desenvolvimento Dirigido a Testes — TDD
  • Integração com dispositivos IoT / ESP32
  • Controle de versionamento (Git / GitHub)

🎯 Objetivos de Aprendizagem

  • Implementar software aplicando engenharia de software e gerência de projetos
  • Integrar aplicações desktop, web e mobile com transparência
  • Utilizar sensores IoT para captura de dados tratados nas aplicações
  • Aplicar linguagens back-end e front-end adequadas a cada plataforma
  • Desenvolver objetos inteligentes com capacidade de interação entre tecnologias
  • Aplicar metodologias ágeis: Projetos, Gamificação, Coding Dojo
  • Construir e consumir APIs robustas e testáveis
  • Entregar software com CI/CD e cobertura de testes adequada
